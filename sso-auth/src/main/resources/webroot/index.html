Secure HTTP API - Vert.x

The mission expands on the REST API Level 0 Booster by securing a REST endpoint using Red Hat SSO. Red Hat SSO implements the OAuth 2.0 specification and uses it to issue access tokens to provide clients with various access rights to secured resources. Securing an application with SSO enables you to add security to your applications while centralizing the security configuration.
